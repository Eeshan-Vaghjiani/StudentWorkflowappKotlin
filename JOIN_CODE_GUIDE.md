## For Group Owners/Admins

### Step 1: Find Your Join Code
1. Open the app
2. Go to **Groups** tab
3. Tap on your group
4. Scroll to the **"Join Code"** section
5. You'll see a 6-digit code (e.g., **ABC123**)

### Step 2: Copy the Code
1. Tap the **"Copy Join Code"** button
2. The code is copied to your clipboard
3. You'll see a confirmation: "Join code copied to clipboard"

### Step 3: Share the Code
Choose any method:

**Option A: WhatsApp**
```
1. Open WhatsApp
2. Select contact or group
3. Paste the code
4. Send message: "Join my study group with code: ABC123"
```

**Option B: SMS**
```
1. Open Messages app
2. Select contact
3. Paste the code
4. Send
```

**Option C: Email**
```
1. Open Email app
2. Compose new email
3. Paste the code
4. Add message and send
```

**Option D: In Person**
```
1. Show them the code on your screen
2. They can type it manually
```

## For People Joining

### Step 1: Get the Code
- Receive the 6-digit code from the group owner
- Example: **ABC123**

### Step 2: Open the App
1. Launch TeamSync Collaboration app
2. Make sure you're logged in

### Step 3: Join the Group
1. Go to **Groups** tab
2. Tap **"Join Group"** button (blue button in Quick Actions)
3. Enter the 6-digit code
4. Tap **"Join"**

### Step 4: Confirmation
- You'll see a success message
- The group will appear in your "My Groups" list
- You can now access group details and participate

## Code Format

### Valid Codes
- **Length**: Exactly 6 characters
- **Characters**: Letters (A-Z) and numbers (0-9)
- **Case**: Uppercase only
- **Examples**: 
  - ‚úÖ ABC123

## Security Features

### Code Protection
- Only admins/owners can see the code
- Regular members cannot view the code
- Prevents unauthorized sharing

### Code Regeneration
If your code gets shared publicly:
1. Open group details
2. Tap **"Regenerate Code"** button
3. Old code becomes **invalid immediately**
4. New code is generated
5. Share the new code with trusted members

### Member Limits
- Maximum 50 members per group
- Code won't work if group is full
- Owner can remove inactive members to make space

## Troubleshooting

### "Invalid join code"
**Possible causes:**
- Code was typed incorrectly
- Code was regenerated by owner
- Group was deleted
- Code is from an inactive group

**Solutions:**
- Double-check the code
- Ask owner for current code
- Make sure code is uppercase
- Try copying/pasting instead of typing

### "Group is full"
**Cause:** Group has reached 50 member limit

**Solutions:**
- Ask owner to remove inactive members
- Owner can increase limit (if implemented)
- Create a new group

### "You are already a member"
**Cause:** You've already joined this group

**Solution:**
- Check your groups list
- The group should already be there

### Can't see "Join Group" button
**Possible causes:**
- Not logged in
- App needs update
- UI issue

**Solutions:**
- Restart the app
- Check you're logged in
- Look in Quick Actions section

## Best Practices

### For Group Owners
1. **Keep codes private** - Only share with people you want in the group
2. **Regenerate regularly** - If code is shared publicly
3. **Use multiple methods** - Email for formal, WhatsApp for quick
4. **Include context** - Tell people what the group is for
5. **Set expectations** - Let members know group rules

### For Members
1. **Save the code** - In case you need to rejoin
2. **Don't share** - Unless owner says it's okay
3. **Join promptly** - Codes might be regenerated
4. **Verify group** - Make sure it's the right group before joining

## Advanced Features

### Multiple Groups
- You can be in multiple groups

### Group Discovery
- Public groups can be found without code

### Code History
- Currently, only one code per group
- Old codes are invalidated when regenerated
- No code history is stored
